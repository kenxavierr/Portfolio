About Me

My name is Kenneth Santiago. I am 22-year old Web Designer based in Quezon City, Philippines.

I worked as a Graphic Designer at Proventa International where I handled marketing collateral such as event banners,layouts,social media ads,event agendas,infographics and event logos using Illustrator,Photoshop and InDesign.

Then after that I worked as a Front End Developer at Spectrum One where I Convert data,ui/ux wireframes or designs to graphical interface for user to view and interact with data through digital interaction using HTML, CSS, Javascript and Bootstrap and managed clean and maintainable code using GIT.

Then after Spectrum One I worked as a Web Designer at Affinity Express Philippines where my responsibilities were developing and mantaining websites using Wordpress and Beaver Builder based on client specifications.
